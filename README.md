# GIS Portfolio: Jose Nieves Paugh

This portfolio showcases my professional expertise in Geographic Information Systems (GIS), spatial analysis, and advanced cartography.

---

### **Professional Overview**

This collection demonstrates technical proficiency in:
* **Spatial Data Management:** Cleaning, integrating, and maintaining diverse geospatial datasets.
* **Advanced Spatial Analysis:** Utilizing geoprocessing tools for pattern recognition, suitability modeling, and network analysis.
* **Cartography & Visualization:** Developing informative maps, interactive dashboards, and web applications.
* **Remote Sensing:** Advanced imagery analysis and classification.
